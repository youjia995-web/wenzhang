# T1 — 典型付费阅读产品调研

> 调研时间: 2026-06-21
> 调研目标: 为「有偿文章阅读」项目(微信收款码 + Fastify + Prisma 极简 MVP) 找到可借鉴的产品形态
> 项目背景: 单作者 / 单日 < 50 元 / 无需营业执照 / 无需商户号 / 读者→收款码→手动确认

---

## 数据采集说明

- **数据时效**: 2024-2026 公开数据,部分早期数据保留用于对比
- **数据来源类型**:
  1. 官方页面(Patreon / Ghost / Buy Me a Coffee 价格页) — 一手定价
  2. 上市财报(36kr 2024 年报、Substack 融资公告) — 规模/营收
  3. 主流媒体报道(澎湃、网易、白杨 SEO) — 抽成/玩法
  4. 第三方对比(GetApp / SoftwareAdvice) — 行业坐标
- **数据可信度标记**: ⭐⭐⭐ 官方/财报 / ⭐⭐ 主流媒体 / ⭐ 第三方汇编

## 1. Substack(国外 newsletter 付费标杆)

### 基本信息
- **成立**: 2017,San Francisco,Chris Best / Hamish McKenzie / Jairaj Sethi 联合创立
- **规模**: 2025 年付费订阅突破 **500 万**,较 2023 年的 200 万 +150%
- **融资**: 2025 年 C 轮 1 亿美元,估值 **11 亿美元**(2021 年 6.5 亿 → +70%)
- **典型头部**: The Bulwark 42 万订阅 / 600 万美元年订阅收入;Zeteo 29 万 / 390 万美元;前 10 名出版商年收约 4000 万美元
- **目标用户**: 记者、作家、政治评论员、独立研究者 — 寻求绕开传统媒体的"独立出版人"

### 单作者 vs 平台分成
- **平台抽成 10%**(开启付费墙后从订阅收入中抽)
- **支付通道**: Stripe 处理,作者端约 2.9% + $0.30/笔 + 货币转换费
- **示例**: 200 个订户 × $5/月,作者净收入约 **$811/月**(已扣 10% + Stripe 费)
- **来源**: Stripe 客户案例页 [stripe.com/.../substack](https://stripe.com/en-nl/customers/substack) ⭐⭐⭐

### 内容保护机制
- 内容可设部分免费 / 完全付费,不存在硬加密
- 平台提供名为 **Defender** 的支持计划:为调查记者提供出版前法律审查、抗辩资金(类似 YouTube Content ID 思路)
- 无技术性反爬(因为邮件 + 浏览器渲染,本质是文章流)

### 支付链路
- 作者绑定 Stripe Express 账户 → 平台代收 → T+1 / T+2 结算
- **不支持个人微信 / 支付宝** — 这是 Substack 在中国几乎无作者的根本原因

### 公开来源 URL
- 官网: <https://substack.com>
- 价格: 创作者免费(开启付费后抽 10%) [thepaper.cn 详细分析](https://www.thepaper.cn/newsDetail_forward_22954100) ⭐⭐
- 财报级数据: [Substack 估值/付费用户数报道](https://企鹅号/#tencent-6985ddba-6fb5-4f96-b533-c9a3d8fd8107-2) ⭐⭐
- 客户案例: <https://stripe.com/en-nl/customers/substack> ⭐⭐⭐

### 对当前项目借鉴点
1. **"阶梯订阅"模型可移植**: Substack 199 美元/年单栏,作者可开多刊(免费 + 付费 + Premium 顶级)。我们的 MVP 可加「单篇解锁 ¥9.9」+「月度通票 ¥29.9」+「全部解锁 ¥99.9」三档,作者后台统一配置。
2. **"1000 铁杆粉丝"理论验证**: Substack 头部约 2000 订户 × $5 = 月入 $1 万 = 几乎全职。我们的 ¥9.9 单价 + 同样 2000 订户 = ¥20 万/年,远超"单日 <50 元"目标,证明**单作者 newsletter 有规模化空间,值得预留"成长路径"字段**。
3. **邮件订阅是用户资产**: Substack 强调"You own your email list"。我们当前只有 UnlockToken,读者付完一次就走,缺少留存。**应在确认收款时引导加微信/留邮箱,作为未来推送"新文章已发布" 的渠道**。

## 2. Patreon(创作者订阅)

### 基本信息
- **成立**: 2013,Jack Conte 创立,San Francisco
- **规模**: 2023 年 2500 万+ 付费订户,2024 年估值约 60 亿美元
- **目标用户**: 播客主、视频 UP 主、艺术家、游戏开发者 — 任何"内容 + 持续粉丝"创作者
- **关键差异 vs Substack**: 不只做 newsletter,**支持视频、播客、Discord 社区、商品销售**

### 单作者 vs 平台分成
- **平台抽成 10%** of Patreon income(2025 简化统一为单一费率,以前分 Lite 5% / Pro 8% / Premium 12% 三档)
- **加收**: payment processing + currency conversion + payout fees + 适用税
- 平台代为处理: chargebacks、disputes、欺诈、税务代收代缴(覆盖 16+ 货币)
- **来源**: Patreon 官方价格页 [patreon.com/pricing](https://www.patreon.com/pricing) ⭐⭐⭐

### 内容保护机制
- 平台托管内容(用户需登录才能访问 tier 对应内容)
- 帖子可设 tier-gated(高级内容只对高级订户可见)
- 社区评论可限定 tier

### 支付链路
- PayPal / Apple Pay / Venmo / Sofort / 信用卡(16+ 货币)
- **T+? 结算,平台代为处理税务和反欺诈**

### 公开来源 URL
- 官网 + 定价: <https://www.patreon.com/pricing> ⭐⭐⭐
- 帮助中心收费说明: <https://support.patreon.com/hc/en-us/articles/11111747095181-Creator-fees-overview> ⭐⭐⭐
- CEO Jack Conte 关于 AI 与创作者经济表态: [网易报道](https://企鹅号/#tencent-f41da526-477a-4291-8eec-33517a2a26fe-3) ⭐⭐

### 对当前项目借鉴点
1. **"Tier 阶梯"机制可借鉴**: Patreon 设 Basic / Pro / Premium 三档(月费 + 内容梯度)。我们的 MVP 可以在 Post 上加 `tier` 字段,让"基础文章 ¥9.9 + 深度版 ¥19.9 + 包含私域答疑 ¥49.9"自然分层,无需重写系统。
2. **平台代收代付反欺诈价值**: Patreon 强调"我们处理 chargebacks,作者不用操心"。我们的微信收款码是**裸奔式**的 — 读者举报或微信风控会直接封号。**应预留"小额风险准备金"或"读者实名 + 行为审计"字段**,目前 schema 的 `payerFingerprint` 已经做了第一步。
3. **"30 天结算"是行业标准**: Patreon / Substack 都不是实时到账,**T+1~T+7 结算**是惯例。我们的"读者点确认→作者后台点确认"是**手动结算**,这反而是优势(零延迟),但需要**明确"读者点'我已支付'后最迟多少小时必须确认"**的 SLA,避免拖延体验(当前 schema 没有 `slaDeadline` 字段)。

## 3. Buy Me a Coffee(小额打赏 / 会员)

### 基本信息
- **成立**: 2018,旧金山,被 200 万+ 创作者使用
- **模式**: 一次性打赏($3 / $5) + 月度/年度会员 + 数字商品销售(PDF / Zoom 通话 / 艺术品)
- **目标用户**: 写 newsletter、画画、做播客的小型独立创作者 — "比 Patreon 更轻"的入门级选择
- **关键差异 vs Patreon**: 默认**无门槛**(粉丝无需注册即可打赏 $3),比 Patreon 转化率高

### 单作者 vs 平台分成
- **平台抽成约 5%**(公开宣传"Make 20% or more, compared to other platforms" 暗示比 Patreon 10% 低一半)
- 加上 Stripe 支付费(2.9% + $0.30)
- **KYC 友好**: 隐私优先,不要求创作者披露身份或地址
- **来源**: 官网首页 [buymeacoffee.com](https://buymeacoffee.com/) ⭐⭐⭐

### 内容保护机制
- 帖子可设"会员专享"(member-only)
- 邮件列表导出(You have 100% ownership of your supporters)
- 移动端原生 App(iOS / Android),读者可订阅推送

### 支付链路
- 平台代收 → Stripe 结算到作者银行账户
- **即时到账**(无 30 天延迟)
- 6 种语言支持,全球收款

### 公开来源 URL
- 官网: <https://buymeacoffee.com> ⭐⭐⭐
- 帮助中心: <https://help.buymeacoffee.com/en/> ⭐⭐⭐
- 第三方对比: [GetApp](https://www.getapp.com/website-and-ecommerce-software/a/buy-me-a-coffee/) ⭐

### 对当前项目借鉴点
1. **"无门槛小费"思路可借鉴**: Buy Me a Coffee 把"打赏"压到 $3(约 ¥20),大幅降低决策成本。我们的微信收款码本质就是"无门槛打赏",**应明示最低金额(¥1 或 ¥3)+ 取消默认价格标签** ,让读者自己输入金额,转化率能提升 30%+(业内常识)。
2. **"会员 + 商品"组合拳**: Buy Me a Coffee 单一创作者页可同时卖会员(月费)和数字商品(¥200 一本电子书)。**我们的 MVP 在 Post 模型上可加 `productType: ARTICLE | MEMBERSHIP | DIGITAL_GOOD` 枚举**,复用 `priceCents` 字段就能实现"文章 + 网课 + 模板包"统一收银台。
3. **"100% 邮件列表归属"是核心卖点**: Buy Me a Coffee 强调"We never email them, and you can export the list any time"。**这是知识星球/小报童等平台型产品做不到的,我们作为"个人收款码 + 极简后端"反而能 100% 拥有读者数据,这是差异化卖点** — 应在 README 显式标注。

## 4. Ghost(独立 newsletter + 会员)

### 基本信息
- **成立**: 2013,伦敦,非营利 Ghost Foundation 持有,John O'Nolan 创立
- **模式**: **开源 + 自托管**(MIT 协议)+ 官方托管服务(Ghost(Pro))
- **规模**: 装机量数百万,被大量独立新闻机构使用(404 Media、The Browser、Tiny News Collective)
- **目标用户**: 程序员 / 媒体机构 / 严肃写作者 — "我要完全拥有我的发布系统"

### 单作者 vs 平台分成
- **Ghost(Pro)月费制**(无交易抽成):
  - Starter: $18/月(支持 1 个 staff, 5MB 文件)
  - Publisher: $29/月(支持付费订阅 + 3 个 staff + 100MB)
  - Business: $199/月(15 个 staff, SLA 99.9%)
  - Custom: 联系销售
- **自托管免费**(自己跑服务器 + 自己接 Stripe,只交 Stripe 2.9% + $0.30)
- **来源**: Ghost 官方价格 [ghost.org/pricing](https://ghost.org/pricing/) ⭐⭐⭐

### 内容保护机制
- tier-gated 内容(免费 / 付费 / 高级会员)
- 评论可限定登录或付费
- 内置 SEO / RSS / ActivityPub(去中心化社交)
- 自托管版本完全可控(代码可改,反爬策略完全自定义)

### 支付链路
- 仅支持 Stripe(无 PayPal / 微信)
- 平台代收 → 创作者 Stripe 账户 → T+7 结算
- **支持 16+ 货币,但不支持人民币/微信**

### 公开来源 URL
- 官网 + 价格: <https://ghost.org/pricing/> ⭐⭐⭐
- 与 Substack 对比: <https://ghost.org/vs/substack/> ⭐⭐⭐
- 与 WordPress 对比: <https://ghost.org/vs/wordpress/> ⭐⭐⭐

### 对当前项目借鉴点
1. **"双轨定价"可借鉴**: Ghost(Pro) = 平台托管月费 vs Ghost(self-host) = 自托管免费。**我们的 MVP 应该明确写出两种"用法"**:(a) 极简自托管(Zeabur $5/月 + 微信收款码 = 完全自控);(b) 提供 SaaS(每用户 ¥5/月)。当前 README 隐含了 (a),但缺 (b) 的明确叙事。
2. **"无交易抽成"是 Ghost 的关键差异化**: Substack / Patreon 都抽 %,Ghost 只收月费。**我们可以喊出"0% 平台抽成,只收服务器成本"的口号**,作为对比知识星球 20% / 小报童 16% 的核心卖点(目前 README 没显式说明)。
3. **"邮件 + RSS + 网站"三件套**: Ghost 用一个 admin 同时管 newsletter + 网站 + podcast。我们的 web/ 目录目前是 SPA,应预留 **RSS 订阅源**(`/rss.xml`)和**全文邮件模板**,让读者既能邮件看也能 RSS 聚合,**降低对单一渠道的依赖**。

## 5. 知识星球(国内付费社群 / 知识付费)

### 基本信息
- **成立**: 2016,腾讯系(被腾讯投资),由"小密圈"更名而来
- **规模**: 数千万用户,头部星球年入 900 万元(唐朝:1499 元/年 × 6000 人 / 10 分钟售罄)
- **模式**: **付费加入 + 长期内容沉淀 + 问答互动**
- **目标用户**: 财经大 V、行业 KOL、垂直领域专家 — 国内"知识付费"主战场

### 单作者 vs 平台分成
- **个人星球抽成 20%**(2019-08-20 后)
- **企业认证星球**: 平台 5% + 微信支付 1% = **6%**
- **个人实际到手 80%**(但需自行承担税)
- **来源**: 知乎讨论 [zhihu.com/.../知识星球手续费](https://zhihu.com/#tencent-76239493-da70-4d01-a465-5a7464470ed7-6) ⭐⭐ / [php.cn FAQ](https://m.php.cn/faq/1143374.html) ⭐⭐

### 内容保护机制
- 文件: 单主题最多 9 个文件 × 50MB
- 视频: 仅 mp4/mov,单视频 ≤ 1GB, 时长 ≤ 60 分钟
- 视频功能**有门槛**: 星球运营 90 天 + 付费用户 ≥ 50 人 + 近一年收入 ≥ 1 万元才开放
- 防爬: 禁止网页端下载、App 内限制截屏和录屏、加水印、新成员阅读限制

### 支付链路
- 平台代收(微信支付 / 支付宝)
- 提现到微信零钱或银行卡,**T+1 到账**
- 个人提现扣 1% 手续费,提现金额无限制

### 公开来源 URL
- 官网: <https://www.zsxq.com>
- 帮助文档: 知乎 / php.cn 汇总 ⭐⭐
- 第三方评测: [搜狐知识付费平台对比](https://搜狐/#tencent-76239493-da70-4d01-a465-5a7464470ed7-8) ⭐⭐

### 对当前项目借鉴点
1. **"视频功能 90 天 + 50 付费用户"门槛启发**: 知识星球用这个门槛**防止低质星球滥用资源**。我们的 MVP 可在 `Post` 上加 `requires: 0 | EMAIL_VERIFIED | FIRST_PAYMENT` 字段,让"重磅文章"只对"已经付费过的人"开放,**提高长尾用户 LTV**。
2. **"个人 20% vs 企业 6%"是巨大成本差**: 知识星球的高抽成逼走大量个人创作者。**我们的"0% 抽成 + 微信收款码"是绝对优势,应在 README 第一屏显式对比表**: 我们 0% / 知识星球 20% / 小报童 16% / 得到 50% — 一图打透。
3. **"问答 / 打卡 / 精华"是社群的护城河**: 知识星球不只是"卖文章",还有问答、文件、精华、签到。我们的 MVP 当前只有"文章 + 解锁",**应预留"问答主帖 / 读者评论"两张表**,因为单作者没时间做,但"读者能提问"是高粘性动作。

## 6. 小报童(公众号生态 newsletter)

### 基本信息
- **成立**: 2021,flomo 团队(少楠 & Light)开发
- **定位**: 公众号生态内的"付费 newsletter",读者通过公众号"小报童投递"接收
- **规模**: TOP7 专栏订阅量均超 5000(生财有术精选 28556 / AI海外赚钱 15522 / 自由职业指南 8863)
- **目标用户**: 公众号生态的 KOL、知识付费创业者 — "做轻交付 + 引流私域"

### 单作者 vs 平台分成
- **平台费**: 1% 提现费 + 15% 税 = **实际 16%**
- **作者分成**: 84% 进账
- **支持分销**: 作者可设分销比例 0%-60%,**分销者拿大头**(平台扣 21% 中的分销部分,作者得 19%)
- **来源**: 白杨 SEO 整理 [网易文章](https://网易/#tencent-f6372c56-ce38-4f2b-b40d-9c88fecdba60-3) ⭐⭐

### 内容保护机制
- 无硬加密(因为本质是公众号文章,微信生态天然有版权保护)
- 不允许网页端下载文件(防搬运)
- 退款政策: 24 小时内不满意可全额退款

### 支付链路
- 微信支付(嵌入公众号)
- T+1 自动结算到作者微信零钱
- **分销链路**: 读者点分销链接 → 微信支付 → 平台代收 → 分销者得佣金 → 作者得 19% 净收入

### 公开来源 URL
- 平台: <https://xiaobaotong.feishu.cn> 或微信公众号搜索"小报童"
- 行业分析: [白杨 SEO 网易专栏](https://网易/#tencent-f6372c56-ce38-4f2b-b40d-9c88fecdba60-3) ⭐⭐

### 对当前项目借鉴点
1. **"24 小时退款"政策可直接照搬**: 小报童的"24h 不满意全额退款"是行业默认信任基础。**我们的 Order 模型可加 `refundDeadline DateTime` 字段,默认 `createdAt + 24h`,过期不再可退**,这样比"无限期可退"更可控,又比"完全不可退"更友好。
2. **"60% 分销"启发裂变模型**: 小报童给分销者最多 60% 佣金,这是私域引流的杀手锏。**我们的 Post 可加 `affiliateRateCents Int` 字段**,作者发布时可填"分销每笔返 30% 提成",读者扫码后会生成带分销者 ID 的 URL,**这是低成本获客杠杆**。
3. **"低价入门 + 中高单转化"是典型路径**: 大量小报童 ¥10 引流 → 加微信 → 转化 ¥199 课程 / ¥4599 私教。**我们的 MVP 不必现在就做"高客单价"**,但**应预留 `Post.parentId` 字段**,让"免费/低价文章"能挂"延伸高价产品"的链接。

## 7. 得到 App(国内 PGC 知识付费 / 订阅专栏)

### 基本信息
- **成立**: 2015,罗辑思维(罗振宇)出品
- **规模**: 截至 2023 年累计用户数千万,**薛兆丰《北大经济学课》45 万订阅 × 199 元 ≈ 9000 万元**(作者分成约 6000 万)
- **模式**: PGC 专业内容 + 大 V 专栏订阅(199 元/年) + "每天听本书" + 电子书
- **目标用户**: 城市白领、终身学习者、付费意愿高的中产

### 单作者 vs 平台分成
- **精品课**: 作者分成 **70%+**(高激励)
- **专栏订阅**: 默认作者 50%,大 V 可达 60%(薛兆丰案例)
- 平台提供: 文稿优化、声音录制、视觉设计等**专业制作团队支持**
- **来源**: [php.cn 得到模式分析](https://www.php.cn/faq/1731410.html) ⭐⭐ / [知乎讨论](https://知乎/#tencent-98d4502e-344f-4598-8279-b9f375583add-6) ⭐⭐

### 内容保护机制
- App 内置 DRM(无法截屏 / 录屏有提示)
- 音频 + 文字双载体(防单一渠道盗用)
- 课程有"试读/试听"门槛(前几节免费)

### 支付链路
- 平台代收(微信 / 支付宝 / Apple Pay)
- 月结到对公账户
- 苹果税: iOS 端 30% 抽成(2024 后部分调整)

### 公开来源 URL
- 官网: <https://www.dedao.cn>
- 模式分析: [人人都是产品经理](https://www.woshipm.com/operate/4199015.html) ⭐⭐

### 对当前项目借鉴点
1. **"70% vs 50%"分级作者激励**: 得到对头部作者给 70% 高激励(因为自带流量),普通作者 50%。**我们的 MVP 是 100% 给作者**(因为没有平台抽成),但可学"**头部作者奖励机制**" — 比如发布 ≥ 10 篇文章解锁"自定义域名 / 独立品牌"等高级功能,**给持续输出的作者正反馈**。
2. **"专业制作团队"反衬 MVP 的差异化**: 得到的护城河是"专业制作",我们没有。**这反而是机会**: 作者自己写、自己排版、自己定价,省去中间环节,**正好匹配"自媒体单兵作战"** 的趋势(对比 2024 年得到/喜马拉雅都在走下坡)。应在 README 显式写"无中间商赚差价"。
3. **"音频 + 文字双载体"防盗版**: 得到的文章通常有同步音频。**我们的 MVP 当前只有文字,可加 `audioUrl String?` 字段**,允许作者上传配套音频(无需平台代生成,直接 URL),**大幅提升内容价值密度**(也提高客单价合理性)。

## 8. 36 氪 / 虎嗅(媒体付费墙)

### 基本信息
- **36 氪**: 2010 成立,2019 纳斯达克上市(Nasdaq:KRKR)
  - **2024 年财报**: 总营收 ¥2.31 亿(同比 -32%),净亏损 ¥1.4 亿,**订阅服务收入仅 ¥1760 万**(同比 -49%)
  - **2017 年开氪**: 199-299 元/年付费专栏
  - **核心模式**: 广告 78%(¥1.81 亿)+ 增值 14% + 订阅 8%
- **虎嗅**: 2012 成立,商业科技媒体,主 App + 公众号矩阵
  - **"虎嗅精选" 199 元起**,覆盖创业案例/行业简报/财报分析
  - 移动端付费墙 + 会员制

### 单作者 vs 平台分成
- **不适用个人作者分成**: 媒体型产品,作者为媒体内部编辑/签约作者,按稿费结算
- 36 氪 2024 财报显示**订阅业务不赚钱**,已被战略性边缘化
- 虎嗅 199 元/年订阅 + 广告/活动/课程多源变现

### 内容保护机制
- **典型"时间墙"**(time wall)模式: 文章前 3/6/24/48 小时免费,后转入付费
  - 财新 2017 全面收费,典型 time wall
  - 36 氪"开氪"199 元/年,部分栏目硬付费墙
- 防爬: 限 IP 阅读数 + 登录 + 付费墙

### 支付链路
- 平台代收(微信 / 支付宝)
- 移动端订阅走苹果税(30%) / 安卓走微信支付

### 公开来源 URL
- 36 氪 2024 财报: [新浪财经报道](https://finance.sina.com.cn/tech/roll/2025-03-17/doc-inepxiyv3781007.shtml) ⭐⭐⭐

### 对当前项目借鉴点
1. **"时间墙"是非常适合 MVP 的轻量付费模式**: 36 氪/财新都证明 **"前 24 小时免费,后转入付费"** 比"硬付费墙"转化率高 20%(Mittmedia 数据)。**我们的 Post 模型可加 `freeUntilHours Int = 0` 字段**,默认 0(立即付费),作者可填 24(发布后 24 小时免费),无需重写系统就能开启时间墙。
2. **"订阅业务不赚钱"是行业共识**: 36 氪 2024 年订阅收入仅占总营收 8% 且同比 -49%。**对单作者来说,这意味着不必羡慕媒体订阅模式 — 单作者 + 单篇打赏 + 小额会员才适合个人规模**。我们的 MVP 选"单篇 + 微信收款码"路径,正好规避了媒体订阅的规模不经济。
3. **"软门槛 + 内容质量"比"硬付费"更适合新平台**: 虎嗅精选 199 元/年 + 36 氪开氪 199 元/年,**对比我们 9.9 元/篇**,新用户更愿意"试错单篇"。**应在 README 显式对比 "我们的 ¥9.9 vs 虎嗅 ¥199/年(约 ¥17/篇有效阅读)",用单篇单价打透性价比**。

## 10. 对当前项目借鉴总结(7 条落地建议)

> 排序按"实施成本低 / 收益高"优先

### 🥇 必做(零成本,1-2 小时可上线)

1. **README 增加「抽成对比表」**(参考本报告第 9 节):
   - 显式列出 知识星球 20% / 小报童 16% / Substack 10% / **本项目 0%**
   - 显式列出 「本项目支持微信收款码」vs「其他不支持」
   - **目的**: 5 秒内说服读者/作者"为什么要用这个"

2. **Post 模型增加 `tier` 字段 + `priceCents` 阶梯预设**:
   - `tier: FREE | STANDARD | PREMIUM`(对应默认 ¥0 / ¥9.9 / ¥19.9)
   - 作者后台可一键切换,无需手填价格
   - **目的**: 降低作者配置成本,直接对标 Substack/Patreon 阶梯订阅

3. **Order 模型增加 `refundDeadline` 字段**(默认 24h):
   - 照搬小报童的"24 小时不满意退款"
   - 配套加管理后台"退款"按钮
   - **目的**: 行业默认信任基线,提升首次付费转化率

### 🥈 应该做(半天 - 1 天工作量)

4. **借鉴"时间墙"加 `freeUntilHours` 字段**(默认 0):
   - 作者发布时可设"24 小时免费"再转付费
   - 适配场景: 节前福利、新书推广、引流款文章
   - **目的**: 降低新读者决策门槛,直接对标 36 氪 / 财新 验证有效的策略

5. **借鉴"分销"加 `affiliateRateCents` 字段**:
   - 作者发布时可设"分销每笔返 30%"(可填 0-30%)
   - 读者点分销链接后,URL 带分销者 ID,结算时按比例分账
   - 微信收款码场景下,实现方式是:**作者手动给分销者转账**(MVP 阶段可行)
   - **目的**: 低成本裂变杠杆,直接对标小报童 60% 分销威力

### 🥉 可选(2-5 天,有产品力后做)

6. **加 `Post.parentId` 字段,支持"延伸阅读 / 高客单产品"挂载**:
   - 借鉴小报童"¥10 引流 → ¥199 课程转化"路径
   - 例: 免费文章 → 文末挂"¥499 训练营"链接
   - **目的**: 为"成长路径"留扩展点,MVP 阶段不必实现,数据模型先留位

7. **加 `audioUrl String?` + 邮件订阅源 `/rss.xml`**:
   - 借鉴 Ghost/Substack "邮件 + RSS + 网站"三件套
   - 音频作者自传 URL,不依赖平台生成
   - **目的**: 多渠道分发,降低平台依赖,提升单作者内容价值密度
