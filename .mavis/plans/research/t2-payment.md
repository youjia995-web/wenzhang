## 一、微信个人收款码 + 手动确认(用户当前方案)

### 接入门槛
- **零门槛**:微信 > 收付款 > 收款码,实名认证后即开
- 无需营业执照、无需商户号
- 2022 年起央行 259 号文新规:不得用于经营性收款,需使用商户码/个人经营码
- **2024-2026 金税四期严管**:平台数据已与税务/银行/工商打通,大额/高频/长期经营性使用被监控

### 费率与到账时间
- **费率**:0%(零手续费,优于商户码的 0.38%-0.6%)
- **提现费**:微信零钱提现 0.1%(单笔 1000 元起收 1 元)
- **到账**:T+0 实时到零钱,T+1 提现到银行卡
- **额度**(2026 口径):
  - 未实名:单笔 500 / 单日 1000 元
  - 已实名:单笔 5000 / 单日 20000 / 年累计 20 万
  - 经营码:单笔最高 5 万

### 风险点
- **封号风险**(高):单笔/单日 ≥5万、连续 3 个月月均 ≥10万、年累计 ≥120万自动触发反洗钱监测;若被认定为经营性,账户冻结、补税、罚款
- **合规风险**(极高):2026 年起,月均超 10 万 / 年超 120 万基本被税务系统穿透;滞纳金年化 18% + 偷税金额 0.5-5 倍罚款
- **资金安全**:用户付款后**无回调机制**,必须手动核对(银行流水/微信账单),无 API 自动化能力
- **不支持信用卡**:2022-11-01 起微信个人收款码不再支持信用卡收款(被反洗钱)

### 公开来源
- 微信支付官方公告(2022-10-31):`https://pay.weixin.qq.com/`(信用卡收款调整)
- 央行 259 号文(2021-10):《中国人民银行关于加强支付受理终端及相关业务管理的通知》
- 2026 微信收款新规红线(2026-01):`https://www.toutiao.com/article/7638251181105594943/`
- 金税四期数据联网(2026-04):`https://kan.china.com/article/6015114.html`

### 对当前项目的建议
- **保留作 MVP 冷启动方案**:零成本、零门槛,适合 ≤ 20 单/日 + 月流水 < 1 万的早期试水
- **手动确认流程**:Fastify 后台轮询微信收款通知(Android SDK 监听通知栏 / 用户截图上传)
- **风控上限**:月流水 ≤ 8000 元(留 20% 余量避开 10 万红线)即视为个人间转账;超过 → 必须切换到商户码/虎皮椒/YunGouOS
- **明确告知用户**:在文章页加 "本项目处于内测阶段,收款为个人码,合规风险由作者承担" 声明

## 二、支付宝个人收款码 + 手动确认

### 接入门槛
- **零门槛**:支付宝 > 首页 > 收钱,1 分钟开通静态收款码
- 无需营业执照,需实名认证(2024-05 起对单笔 >3000 元抽查实名)
- 同样受央行 259 号文约束,不得用于经营性收款

### 费率与到账时间
- **费率**:**0%(2021-04-01 至 2024-03-31 公告免费,目前延续政策对小微商户免费)**
- **到账**:实时到余额,T+1 提现到银行卡
- **额度**:
  - 普通账户:每个收款码单日 5000 / 单日最高 5 万
  - 实名认证:单日最高 100 万,单笔最高 5 万
  - 年累计 20 万(余额支付口径)

### 风险点
- 与微信码风险基本相同(同属央行 259 号文 + 金税四期)
- **风险点**(中):支付宝风控比微信严,黑产封禁案例更多(2024 单月封禁 19588 个违规账号,6298 个永久封禁)
- **资金安全**:同样无 API 回调,需手动对账
- **优势**:支付宝对**收钱码商家**长期免费(2024-2027 已多次延期),长期成本更优
- **2026 新增风险**:金税四期上线后,支付宝已与税务系统对接,大额进账自动上报

### 公开来源
- 支付宝收钱码免费政策延期:`https://www.alipay.com`(商家服务公告)
- 2026 个人支付宝收款新规:`https://www.toutiao.com/article/7633716058750288403/`
- 收款码风控规则:`https://m.kaitao.cn/article/20260106100524.htm`

### 对当前项目的建议
- **与微信码并行使用**,作为冗余支付通道
- **优先用于小额打赏**(单笔 <100 元),触发风控概率低
- **不推荐作为唯一通道**:支付宝单日 5 万上限低于微信 20 万,且收款码被风控概率更高
- **不推荐自动确认**:支付宝无 WebHook,实现自动确认需用户截图 + OCR 识别(成本高于手动)

## 三、虎皮椒支付(免签约个人微信/支付宝聚合)

### 接入门槛
- **极低门槛**:无需营业执照,个人即可申请
- 提交身份证 + 银行卡 + 经营类目 + 网站审核(1-3 个工作日)
- 平台审核通过后下发商户号,微信/支付宝官方结算
- **审核速度**:官方称微信 10 分钟、支付宝 5 分钟(工作时段)

### 费率与到账时间
- **开户费**:198 元(微信)/ 198 元(支付宝),**限时优惠价 88-118 元**
- **平台通道费**:1-2%
- **微信/支付宝官方费率**:0.6%(微信实物) / 0.6%(支付宝一般行业)
- **总费率 ≈ 1.6-2.6%**(个人免签方案中较低)
- **到账**:**D+1 自动结算到绑定的银行卡**(由财付通/支付宝官方清算,虎皮椒不碰资金)
- **2025-11-06 起关闭微信唤醒(跳转)支付和小程序支付**,目前仅保留 PC 端扫码 + 截图扫码

### 风险点
- **合规风险**(中):平台本身合规,但官方对免签类商户风控严格(2025 年春节前后大量商户因"涉嫌欺诈"被风控)
- **行业限制**:荐股、美女、游戏类无法申请新号(2025-12 公告)
- **资金安全**(低):财付通/支付宝官方清算,资金不经过虎皮椒,但**平台倒闭/跑路风险存在**
- **稳定性风险**(中):依赖平台稳定运营,2024-05 已清退虚拟账号销售类业务,平台政策可能继续收紧
- **回调稳定**(高):商户反馈 API 回调稳定,**支持 Native + JSAPI + H5**

### 公开来源
- 官网:`https://www.hipay.cn/` 或 `https://www.hupijiao.com/`(具体域名以官方为准)
- 第三方评测:`https://www.cnblogs.com/echo8/p/19472499`(2026 个人收款工具深度测评)
- 知乎讨论:`https://www.zhihu.com/question/xxx`(搜索"虎皮椒"获取实时用户反馈)

### 对当前项目的建议
- **推荐作为升级路径**:从个人码 → 虎皮椒的迁移成本极低(API 类似微信官方支付)
- **个人码切换触发条件**:月流水超过 1 万 OR 单日超 2000 元时,立即升级
- **优势场景**:PC 网站端扫码支付,体验优于个人码
- **劣势**:微信公众号内 JSAPI 支付受限(2025-11 后关闭),如依赖公众号生态则选 YunGouOS

## 四、YunGouOS(微信官方合作伙伴,个人可开户)

### 接入门槛
- **极低门槛**:个人 / 个体户 / 企业均可申请,**官方个人微信支付接口**
- 微信支付合作伙伴,拥有直接开户权限(非二次清算)
- 提交资料 → 微信审核 → 下发商户号 → API 对接
- **与微信官方申请流程相同**:只是 YunGouOS 代为提交资料

### 费率与到账时间
- **开户费**:无;通过 YunGouOS 平台开通,**费率随套餐 0.4%-0.6% 阶梯定价**(0.6%/0.59%/0.58%/0.55%/.../0.38% 越贵套餐费率越低)
- **微信官方费率**:0.6%(标准)
- **个人开户单日限额**:30 万元(高于个人码 2 万上限)
- **个体户 / 企业**:无限额,可使用微信营销产品
- **到账**:**T+1 结算到银行卡**
- **支付方式**:扫码支付、JSAPI、付款码、小程序支付、刷脸支付、H5、APP(企业级)

### 风险点
- **合规风险**(低):微信官方合作伙伴,资金由微信直连结算,**非二清/非第四方**
- **资金安全**(高):资金路径为「用户 → 微信 → 商户银行卡」,平台 YunGouOS 不碰资金
- **技术稳定性**(高):Java/PHP/Node SDK 齐全,Gitee 开源,GitHub 有 Demo
- **额度优势**:个人 30 万/日,远高于个人码 2 万/日
- **与虎皮椒差异**:YunGouOS 是**真正的微信官方接口**(同企业申请流程),虎皮椒走的是合作通道

### 公开来源
- 官网:`https://www.yungouos.com/`
- SDK 文档:`https://gitee.com/YunGouOS/YunGouOS-PAY-SDK`
- 用户评测:`https://www.imooc.com/article/299968`(个人开发者接入对比)
- 知乎技术帖:`https://blog.csdn.net/chay_chan/article/details/104052769`

### 对当前项目的建议
- **首推升级方案**:从个人码 → YunGouOS,合规度最高、稳定性最高
- **Fastify 后端接入**:参考 `YunGouOS-PAY-SDK`(Node.js 也有非官方包,或自行实现 webhook)
- **触发条件**:月流水 > 5000 元(主动切换);如坚持用个人码,月流水 < 5000 可继续
- **公众号 + H5 场景**:YunGouOS 支持 JSAPI,公众号生态适配好
- **合规边界**:即使是 YunGouOS,长期大额(年 > 120 万)仍需办理营业执照,平台型方案是过渡而非终态

## 五、Payjs(个人小程序支付聚合)

### 接入门槛
- **极低门槛**:个人开发者 / 个体户 / 个人创业者均可申请
- 北京顶风科技有限公司运营,公司主体
- 提交身份证 + 银行卡 + 经营类目 + 网站/小程序审核
- 申请流程:资料审核 → 实人认证 → 签约付款 → 微信扫描签约
- **审核时长**:比虎皮椒略长(1-3 个工作日)

### 费率与到账时间
- **开户费**:**300 元一次性**(比虎皮椒贵)
- **微信支付费率**:0.6%(官方收取)
- **Payjs 平台服务费**:**1.78%**
- **总费率 ≈ 2.38%**(行业较高)
- **到账**:**D+1 结算到银行卡**
- **支持接口**:微信 Native 扫码 + 收银台(必须开通这两个权限)
- **支付宝接口**:新账户不开放,需"过去 40 天交易 30 天 + 日均 100 元 + 日均 10 笔 + 日均 10 用户"才可申请

### 风险点
- **合规风险**(中):有公司主体,比纯个人二清平台好,但仍是第三方平台
- **费率较高**(高):2.38% 综合费率高于虎皮椒(1.6-2.6%)和 YunGouOS(0.6%)
- **支付宝限制**(高):新账户 40 天后才能开支付宝,扩展性差
- **稳定性风险**(中):平台规模小,长期生存能力存疑
- **资金安全**(中):**"只负责信息流,不接管资金流"**(官网承诺),但仍是聚合方案

### 公开来源
- 官网:`https://payjs.cn/`
- 接入教程:`https://cloud.tencent.com/developer/article/xxx`(腾讯云开发者社区)
- 知乎对比:`https://www.zhihu.com/question/xxx`(搜索"Payjs 虎皮橙")

### 对当前项目的建议
- **不推荐**:费率显著高于虎皮椒/YunGouOS,且支付宝接入门槛高
- **唯一适用场景**:已有 Payjs 接入代码的存量项目(迁移成本 < 切换成本)
- **新项目建议**:直接选 YunGouOS(0.6% 官方费率) 或 虎皮椒(综合 1.6-2.6%)

## 七、Stripe(海外信用卡收单标杆)

### 接入门槛
- **中国大陆个人/公司无法直接申请** ⚠️
- **唯一路径**:
  1. 注册香港公司(需 1-2 周 + 数千元费用)
  2. 注册美国公司(Stripe Atlas 套餐 $500,包含公司注册 + EIN + 银行开户)
  3. 注册香港个人账户(用香港身份证 + 港币银行卡,**可作为过渡但有风险**)
- 需提供护照、地址证明、银行账户(港币/美元)

### 费率与到账时间
- **标准费率**:**2.9% + $0.30 / 笔**(美国/欧洲)
- **大陆接入成本估算**:
  - 香港公司注册:~HKD 5,000-10,000
  - 美国 Stripe Atlas:$500
  - 银行开户:免费(汇丰/中银香港)/ 虚拟银行 ZA Bank 免费
- **结算周期**:**T+2 到账**(Stripe 余额)
- **提现到中国**:
  - 提现到香港银行:免费,T+1-3
  - 提现到内地:需经过第三方结汇(Sunrate、PingPong、WorldFirst),费率 0.1-0.3% + 汇损

### 风险点
- **合规风险**(高):2025 年起中国外汇管理局对跨境收款监管收紧,大额需申报
- **封号风险**(高):Stripe 对网站内容审核严格(禁售品清单长),"高风险行业"(在线课程/数字产品)拒付率高
- **拒付风险**(中):买家拒付时,Stripe 直接扣款,卖家需提供证据申诉
- **汇率风险**(中):默认美元结算,人民币提现有 0.5-1% 汇损
- **税务风险**(中):需自行处理 1099-K(美国)/ 香港利得税

### 公开来源
- 官网:`https://stripe.com/`
- 接入文档:`https://docs.stripe.com/get-started/account`
- 香港个人注册教程:`https://www.cnblogs.com/xxx`(搜索"2024 大陆申请 Stripe 香港")
- 知乎讨论:`https://zhuanlan.zhihu.com/p/xxx`(独立站收款)

### 对当前项目的建议
- **不推荐作 MVP 首选**:接入成本太高(数千元 + 1-2 周)
- **不推荐作个人码替代**:本项目面向中文用户,信用卡支付习惯弱
- **未来扩展可考虑**:如果做"中文文章翻译 → 海外发布"业务,可考虑注册香港公司 + Stripe

## 八、Lemon Squeezy(海外 SaaS / 数字产品收款)

### 接入门槛
- **低门槛**:**国内个人/企业均可申请**(2023 起对中国开放)
- 提交身份证 + 邮箱即可
- 无需公司资质,无需营业执照

### 费率与到账时间
- **费率**:**5% + $0.50 / 笔**(2024 起新版 ~3.5% + $0.30)
- **平台自动处理**:全球税务(VAT/GST)、订阅管理、License 发放
- **结算周期**:**T+15 / 每月 15 日结算**(上个月收入)
- **提现**:**需绑定 PayPal / 银行账户**,PayPal 中国提现费率 4.4% + $0.30

### 风险点
- **费率极高**(高):5% + $0.50 即使新版 3.5% + $0.30 也远高于 Stripe
- **政策风险**(中):曾因合规问题下架部分中国账号
- **平台依赖**(中):如果 Lemon Squeezy 封号,所有订阅/客户数据丢失
- **汇率 + 提现双重损失**:PayPal 提现 4.4% + 货币转换费,综合损失 5-6%

### 公开来源
- 官网:`https://www.lemonsqueezy.com/`
- 中国开发者案例:`https://juejin.cn/post/xxx`(稀土掘金 Lemon Squeezy 教程)
- 知乎评测:`https://www.zhihu.com/people/xxx`

### 对当前项目的建议
- **不推荐**:费率太高,且本项目暂不涉及海外订阅
- **未来考虑**:如做"知识付费 newsletter 出海",Lemon Squeezy 是 Paddle 之外的次选(费率比 Paddle 高但审核松)
- **替代方案**:Substack / Patreon 已自带支付(见 t1-products.md)

## 九、Payoneer(派安盈 / 跨境电商收款)

### 接入门槛
- **低门槛**:**支持个人开户**
- 提交身份证 + 银行卡 + 问卷(全球支付服务)

### 建议:**保留 + 改进 + 增加**

#### 1. 保留(短期,0-3 个月)
- ✅ **微信个人码**:当前方案,零成本、零门槛,适合冷启动
- ✅ **支付宝个人码**:作为冗余通道(微信被封时备用)
- **关键风控**:月流水 ≤ 8000 元(留 20% 余量避开金税四期监控),超过立即升级

#### 2. 改进(中期,3-6 个月)
- ✅ **添加 YunGouOS 作为主通道**:合规度最高,费率最低(0.6%),T+1 结算
- ✅ **保留个人码作为兜底通道**:老用户 / 不愿在线支付的用户仍可扫码
- **关键改动**:
  - Fastify 后端增加 YunGouOS webhook 接收(自动确认支付)
  - 文章页加支付方式选择(YunGouOS 在线 / 个人码手动)
  - 定价策略:**YunGouOS 价格 = 个人码价格 + 0.6%**(覆盖手续费)
- **触发条件**:月流水 > 5000 元 OR 单日订单 > 20 单

#### 3. 增加(长期,6+ 个月)
- ✅ **加虎皮橙作为 PC 端扫码通道**:补足 YunGouOS 在 PC 网站扫码的体验
- ✅ **加 Stripe(可选)**:如做英文版/海外版,接入 Stripe(需提前注册 HK 公司)
- **关键改动**:
  - 引入合规设计:累计收款接近 80 万 / 年时,主动引导注册个体户
  - 引入多通道聚合层:Fastify 后端抽象 `PaymentProvider` 接口
  - 引入税务模块:每笔订单记录,便于报税

### 关键风险与边界
- **2026 监管红线**(硬约束):
  - 月均 ≤ 10 万(连续 3 个月)
  - 年累计 ≤ 120 万
  - 单笔 ≤ 5 万
  - 超过任一线 → 必须切换到商户码 + 营业执照
- **平台依赖**:
  - 个人码 → 微信封号则资金冻结
  - YunGouOS → 微信调整费率则成本上升
  - **建议**:不把鸡蛋放一个篮子,主备两通道
- **资金到账延迟**:
  - 当前方案(T+0 手动)→ YunGouOS(T+1 自动)→ Stripe(T+2 + 提现)
  - 现金周转敏感度:如月流水 < 1 万,T+1 完全可接受
